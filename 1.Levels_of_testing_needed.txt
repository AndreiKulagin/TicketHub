                                                 Levels of Software Testing.

1.Unit - test specific lines of code (unit).

2.Integration - test integrated components.

3.System - test entire system.

4.Acceptance - checking the system for compliance with business goals.

For the Task1 we need to use System testing. In this task we do functional testing ,

the main functions of the website as creating a new company, department etc.
